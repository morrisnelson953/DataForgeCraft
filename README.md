# DataForgeCraft - Structured Dataset Generator

DataForgeCraft helps developers and data scientists generate clean, fake data for testing pipelines, APIs, or dashboards.

## Features
- Supports CSV, JSON, and SQL  
- Customizable schema  
- Fast batch output
